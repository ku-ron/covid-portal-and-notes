## ワクチンの名前を分けて呼び分ける提案
<blockquote class="twitter-tweet"><p lang="ja" dir="ltr">今我々がワクチンと呼んでいるものに以下の二つがあるとして、<br><br>1.感染拡大を防ぐ効果があるもの(他者にうつす可能性を減らすことが証明されたもの)<br>2.感染拡大を防ぐ効果が証明されていないもの<br><br>1のみ「ワクチン」と呼んで、2には別の名前を付けてそれで呼ぶことを必須にするのはどうでしょうか。 <a href="https://t.co/a2OqK0HhRY">https://t.co/a2OqK0HhRY</a></p>&mdash; 岡田龍太郎／クーロン (@suck_a_sage) <a href="https://twitter.com/suck_a_sage/status/1997109090796163543?ref_src=twsrc%5Etfw">December 6, 2025</a></blockquote> <script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>

```tweet
インフルエンザや新型コロナのワクチンの認可されたときの薬剤としての説明には、「他者への感染（伝播）を防ぐ／下げる」効果は、正式な効能・効果としては記載されていないわけですよね。

しかし、我々がワクチンと呼んでいるものには、普通その効果を期待しているのだと思うのです。
```

```tweet
ワクチンには普通その効果を期待しているからこそ、特に説明しなければ、多くの人は他者への感染を防ぐ効果があるものだと考えます。だからこそ、接種しない人へのバッシングなどが発生しているわけです。ということは、その効果がないことは特筆すべきことであり、黙っていることを許すべきではない。
```

```tweet
多くの医療者は、「薬剤の説明に書かれてないけど、まあ証明できてないだけでぼんやりその効果もあるんじゃないかなあ」みたいな甘い判断をしているのではと想像するのですが、例えば症状の緩和が免疫寛容によるものであったりすれば、より感染拡大に寄与している可能性だってあります。
```

```tweet
だから薬剤の承認時の説明以上のことはプラスもマイナスも期待するべきではない。そして、ワクチンという言葉には感染拡大を防ぐものという社会的イメージがあるから、そうでないものは特別説明すべきだけど、それを毎回追加でするのは大変。

そこで、そもそも名前を分けるのはどうかという提案です。
```

## 新型コロナウイルスの株の命名法

<blockquote class="twitter-tweet"><p lang="ja" dir="ltr">新型コロナウイルスの株の命名がよく分からなかったのでChatGPTに聞いてまとめてもらいました。<br><br>①<br>新型コロナ（SARS-CoV-2）の「株名」が分かりにくい理由は、命名体系が1つではないから。<br><br>系統名（研究用）：Pango系統名<br>一般向け呼称：WHOのギリシャ文字<br><br>が併存している。</p>&mdash; 岡田龍太郎／クーロン (@suck_a_sage) <a href="https://twitter.com/suck_a_sage/status/2000075666818216198?ref_src=twsrc%5Etfw">December 14, 2025</a></blockquote> <script async src="https://platform.twitter.com/widgets.js" charset="utf-8"></script>

```tweet
新型コロナウイルスの株の命名がよく分からなかったのでChatGPTに聞いてまとめてもらいました。

①
新型コロナ（SARS-CoV-2）の「株名」が分かりにくい理由は、命名体系が1つではないから。

系統名（研究用）：Pango系統名
一般向け呼称：WHOのギリシャ文字

が併存している。
```

```tweet
②
最初に見つかったいわゆる「武漢株」は、Pangoでは A系統 / B系統 に分かれて、実際にはその下で A.x / B.x として細分化されて管理されていた。

「武漢株」という名前は通称で、WHO名も正式なPango名も存在しない。「始祖株」「初期株」などとも呼ばれる。
```

```tweet
③
Pango系統名は基本的にこう増える：

B → B.1 → B.1.1 → B.1.1.529 …

枝分かれのたびに「.数字」を足す。
進化の系譜をそのまま番号で表すのが目的。
```

```tweet
④ いわゆる武漢株はPango 系統では A 系統および B 系統に属し、その下で A.* / B.* として管理されていた。その後、感染性や免疫逃避などの点で公衆衛生上懸念される系統が出現し、WHO により variant of concern（VOC）という概念が導入され、これらにギリシャ文字の名称が付与されるようになった。
```

```tweet
⑤ オミクロン株の正式なPango名は
B.1.1.529

ただし、これ以上番号を足すと

B.1.1.529.1
のように長くなりすぎる。
そこで導入されるのが エイリアス（省略名）。
```

```tweet
⑥ その結果：
B.1.1.529.1 → BA.1
B.1.1.529.2 → BA.2
となった。
BA は「Bの派生1個目」など意味のあるラベルではない。
単に「長くなりすぎた番号を置き換えるラベル」。

親が B でも AY や BA になることがある （例：デルタ = B.1.617.2 → AY.x）
👉 文字列から系統の意味は読み取れない
```

```tweet
⑦ まとめると：
武漢株：A / B（通称のみでWHO名なし）

オミクロン：B.1.1.529（WHO名あり）
BA.*：B.1.1.529（オミクロン）配下の系統に割り当てられたPangoエイリアス
BA.1、BA.2.86 など：オミクロン株の下位系統

BA は特別な意味の記号ではなく、追跡のための整理番号。
```
